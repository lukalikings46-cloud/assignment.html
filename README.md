# assignment.html# ICU Zambia - Information and Communication University Website

GROUP TWO (2) PROJECT

![ICU Zambia Logo](images/logo.png)

A modern, responsive website for Information and Communication University (ICU) Zambia, featuring a student portal, academic calendar, and comprehensive university information system.

## 🚀 Features

- **Modern Responsive Design** - Fully responsive layout that works on all devices
- **Interactive Academic Calendar** - Search and filter academic events
- **Student Portal** - Secure login system with dashboard
- **Course Registration System** - Interactive course selection
- **Faculty Directory** - Browse faculty members
- **Campus Gallery** - Virtual campus tour
- **Dark/Light Theme Toggle** - Customizable viewing experience
- **Real-time Notifications** - Toast notifications for user actions
- **Progress Tracking** - Visual academic progress indicators
- **Custom Animations** - Smooth scroll, parallax, and hover effects
